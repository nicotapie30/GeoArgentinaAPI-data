Localidades Argentinas

Bienvenido a la API de Localidades de Argentina. Esta te permite acceder a información sobre provincias, departamentos y localidades del país. A continuación, te mostramos cómo puedes empezar a utilizarla.

Guía para Utulizar la API

Antes de comenzar, asegúrate de tener:
  • • Una herramienta para realizar solicitudes HTTP, como Postman o curl.
  • • Conocimientos básicos en HTTP y JSON.

Endpoints Disponibles

Obtener Todas las Provincias

  • URL: /api/provincias
  • Método: GET
  • Descripción:

    Devuelve una lista de todas las provincias

Ejemplo de Solicitud:

curl http://dominio/api/provincia

Ejemplo de Respuesta

Obtener Departamentos de una Provincia

  • URL: /api/provincias/{provincia}/departamentos
  • Método: GET
  • Descripción:

    Devuelve una lista de todos los departamentos de una provincia específica

Ejemplo de Solicitud:

curl http://dominio/api/provincias/Buenos-Aires/departamentos

Ejemplo de Respuesta

Obtener Localidades de un Departamento

  • URL: /api/provincias/{provincia}/departamentos/{departamento}/localidades
  • Método: GET
  • Descripción:

    Devuelve una lista de todas las localidades de un departamento específico

Ejemplo de Solicitud:

curl http://dominio/api/provincias/Buenos-Aires/departamentos/Puelén/localidades

Ejemplo de Respuesta

Obtener Información de una Localidad Específica

  • URL: /api/provincias/{provincia}/departamentos/{departamento}/localidades/{localidad}
  • Método: GET
  • Descripción:

    Devuelve información detallada de una localidad específica

Ejemplo de Solicitud:

curl http://dominio/api/provincias/Buenos-Aires/departamentos/Puelén/localidades/Puelén

Ejemplo de Respuesta
